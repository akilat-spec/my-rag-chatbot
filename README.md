# RAG Chatbot with Multi-Level Namespaces

A Streamlit-based RAG chatbot using Pinecone for vector storage and Groq for LLM inference.

## Features
- Multi-level namespace management
- PDF and Markdown document upload
- Semantic search with embeddings
- Real-time chat interface

## Setup
1. Add your API keys to Streamlit Cloud secrets
2. Deploy to Streamlit Cloud
3. Upload documents and start chatting!
